# backend-site

## Firts steps

Install WSL Ubunto

Install NPM and NVM

Use node:20.11.1

```bash
npm install
```

Install docker and docker compose

Install on VSCode "Prettier - Code formatter" extension

## Run Project Backend

```bash
tsc
npm start
```

## Run Docker Project

```bash
docker compose up -d
```

or

```bash
docker compose up --build --force-recreate --no-deps -d
```

## Stop Docker Project

```bash
docker compose down
```

or

```bash
docker compose down -v
```
